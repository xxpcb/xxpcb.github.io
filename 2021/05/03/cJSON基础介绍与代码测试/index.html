JSONжҳҜд»Җд№Ҳ

JSON(JavaScript Object Notation) жҳҜдёҖз§ҚиҪ»йҮҸзә§зҡ„ж•°жҚ®дәӨжҚўж јејҸгҖӮJSON дҪҝз”Ё JavascriptиҜӯжі•жқҘжҸҸиҝ°ж•°жҚ®еҜ№иұЎпјҢдҪҶжҳҜ JSON д»Қз„¶зӢ¬з«ӢдәҺиҜӯиЁҖе’Ңе№іеҸ°гҖӮ

JSON еҹәжң¬иҜӯжі•

  • ж•°жҚ®д»Ҙ key пјҡvalue (еҗҚз§°/еҖј) зҡ„еҪўејҸеӯҳеңЁ
  • ж•°жҚ®з”ұйҖ—еҸ· , еҲҶйҡ”
  • еӨ§жӢ¬еҸ· {} дҝқеӯҳеҜ№иұЎ
  • дёӯжӢ¬еҸ· [] дҝқеӯҳж•°з»„пјҢж•°з»„еҸҜд»ҘеҢ…еҗ«еӨҡдёӘеҜ№иұЎ

дҫӢеҰӮпјҡ

JSONеҖј(value)зҡ„зұ»еһӢ

  • йҖ»иҫ‘еҖјпјҲtrue жҲ– falseпјү

  • null

  • NUMBER ж•°еӯ—пјҲж•ҙж•°жҲ–жө®зӮ№ж•°пјү

  • STRING еӯ—з¬ҰдёІпјҲеңЁеҸҢеј•еҸ·""дёӯпјү

  • ARRAY ж•°з»„пјҲеңЁдёӯжӢ¬еҸ·[]дёӯпјү

  • OBJECT еҜ№иұЎпјҲеңЁеӨ§жӢ¬еҸ·{}дёӯпјү

    еҪ“з„¶еҜ№иұЎеҶ…йғЁд№ҹеҸҜд»ҘжҳҜж•°з»„пјҢеҰӮ:

    еҜ№иұЎ books еҶ…йғЁжҳҜдёҖдёӘж•°з»„пјҢиҜҘж•°з»„еҸҲеҢ…еҗ«дәҶдёүдёӘеҜ№иұЎ

cJSONеҸҲжҳҜдёӘе•Ҙ

cJSONжҳҜдёҖдёӘеҹәдәҺCзҡ„JSONи§Јжһҗеә“пјҢиҝҷдёӘеә“йқһеёёз®ҖзҹӯпјҢжәҗз ҒеҶ…е®№еҸӘеҢ…еҗ«дёҖдёӘcJSON.hеӨҙж–Үд»¶е’ҢдёҖдёӘcJSON.cжәҗж–Үд»¶пјҢж”ҜжҢҒJSONзҡ„и§Јжһҗе’Ңе°ҒиЈ…пјҢйңҖиҰҒи°ғз”Ёж—¶пјҢеҸӘйңҖиҰҒ#include "cJSON.h"е°ұеҸҜд»ҘдҪҝз”ЁдәҶгҖӮ

cJSONжәҗж–Үд»¶ејҖжәҗйЎ№зӣ®ең°еқҖпјҡhttps://sourceforge.net/projects/cjson/гҖӮ

cJSONзј–зЁӢе®һдҫӢ

дҫӢеӯҗ1

е…Ҳжј”зӨәдёҖдёӘжңҖз®ҖеҚ•зҡ„JSONж•°жҚ®пјҢеҰӮдҪ•дҪҝз”ЁcJSONеә“жқҘи§ЈжһҗгҖӮ

з”ЁдәҺжөӢиҜ•JSONж•°жҚ®пјҢжҲ‘д»¬е…Ҳз”ЁдёҖдёӘcharеһӢзҡ„еӯ—з¬ҰдёІдәҶдҝқеӯҳпјҢжіЁж„ҸпјҢз”ұдәҺJSONеҜ№иұЎзҡ„keyе’ҢvalueйғҪйңҖиҰҒдҪҝз”Ёеј•еҸ·пјҲж•°еҖјжҲ–йҖ»иҫ‘еһӢvalueйҷӨеӨ–пјүпјҢжүҖд»ҘпјҢдҪҝз”ЁCиҜӯиЁҖзҡ„еӯ—з¬ҰдёІжқҘдҝқеӯҳJSONж•°жҚ®пјҢеңЁйҒҮеҲ°еј•еҸ·ж—¶пјҢиҰҒеҠ еҸҚж–ңжқ жқҘиҪ¬д№үгҖӮ

и§ЈжһҗJSONж•°жҚ®ж—¶пјҢйҰ–е…ҲиҰҒи°ғз”Ё cJSON_Parse() иҝҷдёӘеҮҪж•°пјҢе°ҶJSONж•°жҚ®и§Јжһҗдёә cJSON еһӢзҡ„еҜ№иұЎпјҡ

з„¶еҗҺе°ұеҸҜд»ҘдҪҝз”ЁcJSONеә“дёӯжҸҗдҫӣзҡ„еҗ„з§ҚеҮҪж•°жқҘж“ҚдҪңиҝҷдёӘеҜ№иұЎгҖӮжҜ”еҰӮ :

е°ұжҳҜйҖҡиҝҮ cJSON_GetObjectItem() жқҘиҺ·еҸ–JSONж•°жҚ®дёӯзҡ„дёҖйЎ№ж•°жҚ®пјҢе№¶йҖҡиҝҮжҢҮе®ҡkeyзҡ„ж–№ејҸжқҘиҺ·еҸ–еҜ№еә”зҡ„valueгҖӮ

ж•ҙдёӘжөӢиҜ•зЁӢеәҸеҰӮдёӢпјҡ

иҝҗиЎҢз»“жһңпјҡ

дҫӢеӯҗ2

дёҠйқўжҳҜдёҖдёӘжңҖеҹәжң¬зҡ„жөӢиҜ•зЁӢеәҸпјҢJSONж•°жҚ®д№ҹжҳҜзӣҙжҺҘйҖҡиҝҮCиҜӯиЁҖзҡ„еӯ—з¬ҰдёІдҝқеӯҳзҡ„гҖӮ

е®һйҷ…зҡ„дҪҝз”ЁдёӯпјҢJSONж•°жҚ®зҡ„еҶ…е®№иҫғдёәеәһеӨ§пјҢеҶҚдҪҝз”ЁCиҜӯиЁҖзҡ„еӯ—з¬ҰдёІжқҘдҝқеӯҳпјҢеҲҷдјҡеҸҠе…¶йә»зғҰпјҲжғіжғійӮЈдёҖе ҶеҸҚж–ңжқ пјүгҖӮ

жүҖд»ҘпјҢдёҖиҲ¬йғҪжҳҜдҪҝз”Ёж–Үд»¶зҡ„еҪўејҸжқҘдҝқеӯҳJSONж•°жҚ®гҖӮ

еҮҶеӨҮдёҖдёӘеҺҹе§Ӣж•°жҚ®з”ЁдәҺжөӢиҜ•пјҢеҰӮдёӢйқўзҡ„data.txtпјҡ

е…ҲжқҘеҲҶжһҗдёӢиҝҷдёӘJSONж•°жҚ®зҡ„з»“жһ„пјҡ

  • жңҖеӨ–йқўзҡ„еӨ§жӢ¬еҸ·пјҢиЎЁзӨәиҝҷдёӘдёҖдёӘJSONеҜ№иұЎ
  • еҜ№иұЎйҮҢйқўжңү2дёӘж•°жҚ®пјҢkey1жҳҜclassпјҢkey2жҳҜbooks
  • booksзҡ„valueеҸҲжҳҜдёҖдёӘдёӯжӢ¬еҸ·иЎЁзӨәзҡ„ж•°з»„
  • ж•°з»„еҶ…еҸҲеҢ…еҗ«дәҶ3дёӘеҜ№иұЎпјҢжҜҸдёӘеҜ№иұЎжңү3дёӘж•°жҚ®пјҢkey1жҳҜnameпјҢkey2жҳҜauthorпјҢkey3жҳҜpublishing

и§Јжһҗиҝҷз§ҚJSONж–Үд»¶пјҢдҪҝз”ЁиҰҒдҪҝз”Ё fopen() жқҘжү“ејҖиҝҷдёӘж–Үд»¶пјҡ

иҝҷйҮҢиҝҳиҰҒдҪҝз”Ёfseek()е’Ңftell()жқҘи®Ўз®—ж–Үд»¶зҡ„еӨ§е°ҸпјҢзӣ®зҡ„жҳҜз”ЁдәҺеҗҺйқўзҡ„еҲҶй…ҚеҶ…еӯҳж—¶жҢҮе®ҡеӨ§е°ҸгҖӮ

з„¶еҗҺдҪҝз”Ёfread()е°Ҷж–Үд»¶еҶ…еӯҳиҜ»е…ҘеҶ…еӯҳдёӯеҲҶй…Қзҡ„bufдёӯпјҡ

жҺҘзқҖпјҢиҝҳжҳҜдҪҝз”ЁcJSON_Parse() е°ҶJSONж•°жҚ®и§Јжһҗдёә cJSON еһӢзҡ„еҜ№иұЎ

з„¶еҗҺе°ұеҸҜд»ҘдҪҝз”ЁcJSONеә“дёӯжҸҗдҫӣзҡ„еҗ„з§ҚеҮҪж•°жқҘж“ҚдҪңиҝҷдёӘеҜ№иұЎдәҶгҖӮ

иҝҷж¬Ўдјҡз”ЁеҲ°cJSON_GetObjectItem()гҖҒcJSON_GetArrayItem() гҖҒcJSON_GetArraySize()гҖҒcJSON_Print()зӯүеҮҪж•°гҖӮ

жіЁж„ҸжҳҜдҪҝз”Ёе®ҢcJSON_Print()еҗҺпјҢиҰҒдҪҝз”Ёfree()жқҘйҮҠж”ҫгҖӮ
